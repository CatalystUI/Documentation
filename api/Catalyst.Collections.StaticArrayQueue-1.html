Remarks

Allocations should be made using the PeekEnqueue() method, which returns a reference to the next available slot in the queue. Once updated, the Enqueue() method should be fired to increment the write counter by one.

Similarly, the PeekDequeue() method returns a reference to the front-most enqueued slot in the queue. Once read, the Dequeue() method should be called to increment the read counter by one.
