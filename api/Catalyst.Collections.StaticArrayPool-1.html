Remarks

Allocations should be made using the PeekNext() method, which returns a reference to the next available slot in the pool. Once updated, the Allocate() method should be fired, which returns the index of the newly allocated element and increments the count by one.

To release an allocated element, the Release(int) method should be called with the index of the element to release.
