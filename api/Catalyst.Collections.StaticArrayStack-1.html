Remarks

Allocations should be made using the PeekPush() method, which returns a reference to the next available slot in the stack. Once updated, the Push() method should be fired to increment the count by one.

Similarly, the PeekPop() method returns a reference to the top-most element in the stack. Once read, the Pop() method should be called to decrement the count by one.
